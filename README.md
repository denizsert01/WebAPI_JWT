# WebAPI_JWT

A sample ASP.NET Core Web API project demonstrating JWT authentication and service layer architecture.

## 🚀 Features

- JWT Authentication
- Login & Authorization
- Role-based Authentication
- RESTful API Endpoints
- Service Layer Architecture
- Dependency Injection
- Entity Framework Core
- SQL Server Integration

## 🛠️ Technologies

- ASP.NET Core Web API
- C#
- Entity Framework Core
- SQL Server
- JWT (JSON Web Token)
- Dependency Injection

## 📁 Project Structure

- Controllers
- Services
- Models
- Data Access
- Authentication

## ▶️ Getting Started

1. Clone the repository.

```bash
git clone https://github.com/denizsert01/WebAPI_JWT.git
```

2. Update the connection string in `appsettings.json`.

3. Apply the database (if required).

4. Run the project.

## 🎯 Purpose

This project was built to practice implementing JWT authentication, service layer architecture, and RESTful API development using ASP.NET Core.
